This project uses machine learning techniques to predict whether diabetic patients are likely to be readmitted to the hospital within 30 days. The model is trained on the Diabetes 130-US Hospitals Dataset, which contains over 100,000 patient records and 50 clinical features.

The project includes data preprocessing, missing value handling, feature engineering, class imbalance handling using SMOTE, model training, evaluation, and feature importance analysis.

📊 Dataset Information
Dataset: Diabetes 130-US Hospitals Dataset
Source: UCI Machine Learning Repository
Records: 101,766
Features: 50+
Target Variable
1: Patient readmitted within 30 days (<30)
0: Patient not readmitted within 30 days (NO or >30)
